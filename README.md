# Wordle Clone

A ReactJS implementation for the popular word game Wordle.

## Installation

1. Execute ```git clone https://github.com/sruthi-98/wordle-clone.git```.

2. Navigate to newly formed folder by running ```cd wordle-clone```.

3. To install all the dependencies, run ```npm install```.

4. Once everything is installed, execute ```npm start``` to start the application.

**Live Demo** : https://react-clone-wordle.herokuapp.com/
